# TwitchStream
Demo app for Swifty Meetup demonstrating Texture/AsyncDisplayKit

1. Open Terminal.app, type `git clone git@github.com:taufikobet/TwitchStream.git`
2. Type `cd TwitchStream`
3. Type `pod install`, you might need to install Cocoapods first (Type `sudo gem install cocoapods`)
4. Type `open TwitchStream.xcworkspace` to launch Xcode.
5. Build & Run
